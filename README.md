# WeekFood
Proyecto WeekFood para 2ยบ curso de Desarrollo de Aplicaciones Web

## Requisitos
* PHP 7.2
* MariaDB 10.3
* Node.js v8.12.0+

## Instalar dependencias
```
npm install
```

## Iniciar entorno de desarrollo
```
npm run server
```
y en otra terminal
```
npm run cliente
```

## Construir dist/
_(ejecutar desde Git Bash o bash nativo)_
```
npm run build
```
